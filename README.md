## Install
This scripts run on Ubuntu.

```bash
git clone git://github.com/gagomap/nginx_blacklists.git
```
Copy autoclock.sh to /etc/cron.daily

Then run command:

```bash
chmod +x /etc/cron.daily/autoblock.sh
/etc/cron.daily/autoblock.sh
```
